# 과정
## 번외) 만약 수열을 이루는 수에 음수가 포함된다면?
문제에서는 수열을 이루는 수가 양수로 제한된다. 그래서 ans나 max변수의 초기값으로 0을 설정할 수 있었는데, 만약 크기의 제한이 없는 음수도 수열에 포함된다면 어떻게 해야할까 궁금해졌다.  

ans의 초기값을 arr[0]으로 설정하고, max의 초기값을 dp[0] (존재하지 않는다면 arr[0])으로 설정하면 될 것 같다.
