# 과정
## 시간 비교
각 테스트케이스에서 console.log()를 찍는 방법과, 모든 정답을 배열에 담아두고 한 번에 출력하는 방식의 차이를 알고 싶었기 때문에 직접 비교해봤다.

`결과`에 적어둔대로 console.log()의 사용을 최소화 하는게 시간을 줄이는 방법이 된다. 반대로, 정답을 담아두는 배열을 추가적으로 사용하니 메모리 사용량이 많아짐을 확인할 수 있다.
